Understanding Slope

Author

Rahul Gupta

Let \(x\) be a variable that can take any value from \(-\infty\) to \(+\infty\). We can think of \(x\) as the x-axis.
Also, we define: \(y = 2x\).
This means that for any value of \(x\), we simply double it to get \(y\). Example: if \(x\) = 2 then \(y\) = 2\(x\) = \(2 * 2\) = 4.

Note

\(y = 2x\) is a rule we have defined.
In other words, \(y\) is a function of \(x\) because the value of \(y\) depends entirely on the value of \(x\).

We know what \(x\) looks like, it is simply the x-axis. But what does \(y\) look like? Let’s plot it.

import torch
import pandas as pd
import plotly.express as px

x = torch.arange(-100, 100+1, 1)
y = 2 * x

px.scatter(x=x, y=y, height=300).show()


Now let’s think about why that is the case. Let’s investigate a few values of \(x\) and their corresponding \(y\) values. 

y_func = pd.DataFrame({
    "x" : x,
    "y" : y,
})

y_func["x change"] = y_func["x"] - y_func["x"].shift(1)
y_func["y change"] = y_func["y"] - y_func["y"].shift(1)

print(y_func.loc[y_func["x"].isin(range(0,11))].tail(10))
      x   y  x change  y change
101   1   2       1.0       2.0
102   2   4       1.0       2.0
103   3   6       1.0       2.0
104   4   8       1.0       2.0
105   5  10       1.0       2.0
106   6  12       1.0       2.0
107   7  14       1.0       2.0
108   8  16       1.0       2.0
109   9  18       1.0       2.0
110  10  20       1.0       2.0


In our table above, if \(x_1 = 5\) and \(x_2 = 6\), then \(y_1 = 10\) and \(y_2 = 12\).
This implies that if we move one unit along the x-axis (5 to 6), we automatically move 2 units upwards along the y-axis.

Let’s see why that happens. Consider two values of \(x\).

  • If \(x = x_1\), then \(y = 2 \times x_1\)
  • If \(x = x_2\), then \(y = 2 \times x_2\)

Here, \(x_2 > x_1\). Therefore,

  • Increase in \(x\) = Distance moved along the x-axis towards the right
    • Increase in \(x\) = \(x_2 - x_1\)
  • Increase in \(y\) = Due to the movement along the x-axis, how much did we move upwards along the y-axis?
    • \(= y_2 - y_1\)
    • \(= 2x_2 - 2x_1\)
    • \(= 2(x_2 - x_1)\)

Notice that we moved along the y-axis 2 times the distance we moved along the x-axis!

More importantly, we did not specify what the difference between \(x_2\) and \(x_1\) is. This implies that for any increase along the x-axis towards the right, we will always move 2 times upward along the y-axis.

For example, if:

  • \(x_1 = 6.0\), then \(y_1 = 6.0 \times 2 = 12.0\)
  • \(x_2 = 6.1\), then \(y_2 = 6.1 \times 2 = 12.2\)

Increase in \(x\) (movement along the x-axis from \(x_1\) to \(x_2\)): \(= 6.1 - 6.0 = 0.1\)
Increase in \(y\) (movement upwards along the y-axis from \(y_1\) to \(y_2\)):

  • \(= 12.2 - 12.0 = 0.2\)
  • \(= 2 \times 0.1\)
  • \(= 2 \times \text{increase in } x\)

This is why \(y = 2x\) is a line! The line “grows” by 2 for each 1 unit increase in x.

Note

Why is it a line?

The rise (increase in \(y\)) is always the same multiple of the run (increase in \(x\)) for any pair \((x_1, x_2)\) where \(x_2 > x_1\). As the ratio of the rise to the run stays constant, the line never bends. In the next section, we will see what happens when the slope is not constant.

Note

Slope of the line
As we take a step towards the right along the x-axis, we move 2 times as much upwards along the y-axis. Also, this is independent of how big a step we take towards the right side of the x-axis.
If we keep increasing \(x\), the value of \(y\) keeps increasing. This is why we see \(y = 2x\) growing upwards as we move towards the right along the x-axis.

Increase in \(y\) (rise) denoted as \(\Delta y\)
Increase in \(x\) (run) denoted as \(\Delta x\)
\[ \text{Growth or Slope} = \frac{\Delta y}{\Delta x}\]

The above formula is intuitive. For a line, how fast it grows depends on how quickly \(y\) increases when \(x\) is moved slightly to the right along the x-axis. Therefore, the slope or growth is constant for a line (for \(y = 2x\), slope is 2). The higher the value of slope, the steeper the line.

Negative Slope

What would a negative slope imply? It’s actually quite simple.
The negative sign indicates that the line goes downward as the value of \(x\) increases.
Let’s consider another function: \(z = -2x\)

x = torch.arange(-100, 100+1, 1)
z = x * -2
px.scatter(x=x, y=z, height=300).show()

z_func = pd.DataFrame({
    "x" : x,
    "z" : z,
})

z_func["x change"] = z_func["x"] - z_func["x"].shift(1)
z_func["z change"] = z_func["z"] - z_func["z"].shift(1)

print(z_func.loc[z_func["x"].isin(range(0,11))].tail(10))
      x   z  x change  z change
101   1  -2       1.0      -2.0
102   2  -4       1.0      -2.0
103   3  -6       1.0      -2.0
104   4  -8       1.0      -2.0
105   5 -10       1.0      -2.0
106   6 -12       1.0      -2.0
107   7 -14       1.0      -2.0
108   8 -16       1.0      -2.0
109   9 -18       1.0      -2.0
110  10 -20       1.0      -2.0

We observe that:

  1. z decreases by 2 for every 1 unit increase in x
  2. Considering any 2 value of x in the table above, slope = -2

Hence, negative slope indicates negative growth (as x increase, z decreases) and the slope tells us how fast the decrease is (steepness of decrease in the value of z)

TipMain Take away
  1. In the tables above, for each row, we have taken a one step (increased x by one; the next row) and calculated the slope. Slope is constant for straight lines, i.e., for all values of x its the same value.
  2. The slope does not change if we increased \(x\) by a step size other that 1

Consider:

  1. Function: \(y = 5x\)

  2. \(x_1 = a\), then \(y_1 = 5 * x_1\). Let’s calculate slope at \(x_1 = a\)

  3. We increment \(x_1\) by \(c\):

    • \(x_2 = a + c\)
    • Then, \(y_2 = 5 * x_2 = 5(a + c) = 5a + 5c\). In the tables in above sections, \(c = 1\)

  4. Increase in \(x\) = \(x_2 - x_1 = a + c - a = c\)

  5. Increase in \(y\) = \(y_2 - y_1 = 5a + 5c - 5a = 5c\)

  6. \(\text{Slope at } x_1 = \frac{\Delta y}{\Delta x} = \frac{5c}{c} = 5\)

Note:

  • We get a constant slope value. Straight lines have a constant slope.
  • The slope does not depend on value of \(a\) (i.e.,value of \(x\)) and \(c\) (step size). This is why slope is a constant value for all \(x\).
  • The case above holds true for any value of \(a\) (value of \(x\)) and \(c\) (step size)
